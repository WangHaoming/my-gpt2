# 手写数字识别互动演示

一个面向零基础观众的单页演示：在浏览器里**现场训练**一个迷你神经网络（196 → 64 → 10，
约 1.3 万个参数），说明「用带答案的例子训练 → 参数里存下规律 → 对从没见过的新输入做预测」
这一原理，并在结尾桥接到 ChatGPT 类大模型。

## 怎么打开

直接双击（或浏览器打开）`index.html` 即可，无需服务器、无需安装任何东西。
数据文件 `mnist_data.js` 已提交入库，clone 下来就能用。

## 页面结构（5 个步骤）

1. **教材** —— 展示 12000 张带标签的手写数字卡片（机器看到的只是 196 个数字）
2. **训练之前** —— 网络示意图 + 8 道固定考题，随机参数下只能瞎猜（约对 1 道）
3. **开始学习** —— 两种方式一步一步看训练：
   - 「学一步」：单步慢动作，展示 猜测 vs 答案、这一步哪些"旋钮"被拧动（Δ 热力图）
   - 「开始学习」：自动训练约 3~5 秒，实时看 犯错曲线下降、考试成绩上升、
     64 张第一层参数图从随机雪花浮现出笔画模板、步骤 2 的 ✗ 逐个变 ✓
4. **手写测试** —— 观众亲手画数字，网络对**从没见过的**字迹给出预测（泛化）
5. **从认数字到 ChatGPT** —— 静态对比：同样的原理，只是例子、参数、任务的规模不同

## 技术要点

- 纯 vanilla JS，零依赖零构建；前向/反向传播全部手写（`Float32Array` + 循环），
  见 `index.html` 内 `trainStep()` 的逐段中文注释
- 数据：MNIST 降采样到 14×14，类别均衡抽样 12000 训练 + 500 测试，base64 嵌入 `mnist_data.js`
- 训练（步骤 3 的「改进实验室」可逐项开关，训练完自动记入成绩对比表）：
  batch=32、20 遍；默认四项改进全关（基线：6000 张、纯 SGD lr 0.2，约 94%）。
  可逐项开启：教材翻倍（12000 张）、动量 0.9（lr 改 0.05）、学习率线性衰减到 10%、
  ±1 像素随机平移增强；全开约 97%（与 torch 对照一致）
- 手绘预处理复刻 MNIST 规范化：包围盒裁剪 → 等比缩放（长边 10px）→ 质心居中，
  没有这一步，写在画布角落的数字会识别失败

## 重新生成数据（可选）

```bash
python src/math/make_dataset.py           # 下载 MNIST（缓存到 .cache/）并重新生成 mnist_data.js
python src/math/make_dataset.py --check   # 额外用 torch 训一遍同构网络做数值对照（需要 venv）
```

## 讲解建议

现场演示时按页面顺序走即可，两个最打动人的时刻：
1. 步骤 2 → 3：亲眼看着"瞎蒙"的 ✗ 在几秒内一个个变成 ✓；
2. 步骤 4：让观众自己上手写——它认出了一个训练时从未见过的字迹，这就是泛化。
